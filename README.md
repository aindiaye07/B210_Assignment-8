Tuples and Sets

A) Purpose of the Program:
- Write a function that creates a tuple of unique values for “sex”, “smoker”, “day”, “time”, and “size”.

B) Input:
- The program takes the following inputs: Path to a CSV file containing the total bill amount, the tip amount, gender, smoker/non-smoker, the day of the week, the time of day, and party size for a restaurant.

C) Output:
- Function that creates a tuple of unique values for “sex”, “smoker”, “day”, “time”, and “size” is created.

D) Methods:
- Imports
- Initialization of Sets
- Adding Unique Values to Sets
